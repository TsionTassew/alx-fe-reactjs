# Advanced Routing in React with React Router

## Description
This project demonstrates advanced routing techniques in a React application using **React Router**. It includes nested routes, dynamic routes, and protected routes that require authentication. The project is part of the **ALX Frontend ReactJS** curriculum.

---

## Objectives
- Implement routing using React Router
- Create nested routes for structured navigation
- Implement dynamic routes with URL parameters
- Protect routes using authentication logic
- Ensure smooth navigation and redirection

---

## Technologies Used
- React (Vite)
- React Router DOM
- JavaScript (ES6+)
