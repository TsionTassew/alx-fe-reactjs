## Description
This project demonstrates advanced data fetching and server-state management in a React application using **React Query**. It fetches posts from the public **JSONPlaceholder API** and showcases efficient handling of loading states, error handling, caching, and manual data refetching.

This project is part of the **ALX Frontend ReactJS** curriculum.

---

## Objectives
- Integrate React Query into a React application
- Fetch data from a public API
- Handle loading and error states
- Demonstrate data caching and reduced network requests
- Implement manual data refetching

## Technologies Used
- React (Vite)
- React Query
- JavaScript
- JSONPlaceholder API
