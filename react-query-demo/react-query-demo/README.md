
# React Query Demo – Advanced Data Handling

## Project Overview

This project demonstrates advanced data fetching and server-state management in a React application using **React Query**. It fetches posts from the public **JSONPlaceholder API** and showcases efficient handling of API interactions including caching, background updates, loading states, and manual refetching.

This project fulfills the **ALX Frontend ReactJS** mandatory task:
**Advanced Data Handling with React Query**.

---

## Objectives

- Integrate React Query into a React application
- Fetch data from a public REST API
- Handle loading and error states effectively
- Demonstrate caching behavior
- Reduce unnecessary network requests
- Implement manual data refetching
- Improve UI responsiveness

---

## Tech Stack

- React (Vite)
- React Query
- JavaScript (ES6+)
- JSONPlaceholder API
